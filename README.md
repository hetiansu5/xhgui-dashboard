## 简介
如何搭建xhgui可视化面板服务

### 环境要求
- Docker引擎

### 安装和启动

```
git clone https://github.com/hetiansu5/xhgui-dashboard.git
cd xhgui-dashboard
docker-compose up -d
```

### 访问
使用浏览器访问"http://localhost:11000"
